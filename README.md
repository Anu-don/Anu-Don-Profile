# Anurag | Developer Profile

A sleek, responsive, and modern personal portfolio website built to showcase my journey as a developer. This project serves as a central hub for my featured work, technical skills, and professional connections.

## 🚀 About This Project

This portfolio is a single-page web application designed with a focus on clean UI, performance, and responsiveness. It highlights my expertise in full-stack development, game design, and DevOps practices.

### Key Features
- **Responsive Design:** Optimized for all devices, from mobile to desktop.
- **Project Showcase:** Detailed cards for featured GitHub repositories with live demo links.
- **Skills Highlight:** A visual representation of my core technical stack.
- **Interactive Navigation:** Smooth scrolling and active section highlighting.
- **Modern Aesthetic:** Dark theme with custom gradients and professional typography.

### Tech Stack
- **HTML5:** Semantic structure for accessibility and SEO.
- **CSS3:** Vanilla CSS with modern features like CSS Variables, Grid, and Flexbox.
- **JavaScript:** Vanilla JS for dynamic year updates and scroll-based navigation logic.
- **Fonts:** Inter via Google Fonts.
